Bare Metal Architecture (Direct OS)

Distributed setup with a dedicated Hub host and an external PostgreSQL server

System Components

Host A: Centralized Hub

  • FastAPI (Uvicorn)
    Systemd Service (Port 8000)
  • Node.js Runtime
    Local execution of Security Skills
  • Python 3.12 Venv
    CrewAI Orchestration logic
↔

Host B: PostgreSQL

Managed by separate agent

Database Port 5432

Proposed Design for Task 4 (Bare Metal Transition):

  1. Dependency Script: Create a shell script (`setup_host.sh`) to install system-level dependencies (Node.js, LDAP libraries, Python venv).
  2. Systemd Service: Generate a `vbg-hub.service` file to manage the FastAPI process (restart on failure, start on boot).
  3. Logging: Configure Uvicorn to use standard system logging for audit traceability.
