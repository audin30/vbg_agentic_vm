Proposed Approaches for React Web Portal

1. Classic 3-Pane Workspace

Matches the approved mockup. Optimized for single-task focus with immediate context visibility.

  • Sidebar: Searchable query history.
  • Main: Unified stream of Thoughts + Response.
  • Right Panel: Automated Evidence + Risk Gauge.
Best for: Focused Investigations

2. Multi-Tab Command Center

Allows juggling multiple "Active Queries" in browser tabs (internal to the SPA).

  • Top Nav: Tab bar for open queries.
  • Main: Full-width chat area.
  • Collapsible Panels: Evidence/Risk toggled when needed.
Best for: Power Users / High Volume

My Recommendation: Option 1 (Classic 3-Pane)

This layout provides the highest "Context Density" — you see your history, the active streaming chat, and the automated evidence extraction all at once without clicking. It perfectly realizes the "Active Query" terminology we've established.
