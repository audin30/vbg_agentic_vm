Frontend

React 18, Vite, Zustand, Tailwind CSS, Fetch Event Source (SSE).

HITL Feedback Loop

PostgreSQL-backed feedback storage allows agents to learn from human approvals/denials for future autonomous actions.

Backend

Python 3.12, FastAPI, CrewAI, Asyncpg, Jose (JWT), LDAP3.

Database

PostgreSQL 18, JSONB for flexible state, gen_random_uuid for IDs.

Protocols

SSH (Paramiko) for Linux/macOS, WinRM for Windows remediation.
